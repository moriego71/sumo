# SUMO

Repositorio de trabajo para el desarrollo de escenarios de simulación utilizando **SUMO (Simulation of Urban MObility)**.

## Objetivo

Este proyecto forma parte del trabajo de investigación doctoral orientado al diseño y evaluación de una plataforma de **Smart Parking**, utilizando SUMO como entorno de simulación.

El repositorio reúne escenarios de complejidad creciente para validar diferentes funcionalidades antes de su integración con la plataforma principal.
